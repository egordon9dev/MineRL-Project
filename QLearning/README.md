# QLearning
Deep Q-Learning implementation for the MineRLNavigateDense-v0 environment.

## Usage

- To test a preTrained network : run `test.py`
- To train a new network : run `train.py`

## Dependencies
Trained and tested on:
```
Python 3.6
PyTorch 1.0
NumPy 1.15.3
gym 0.10.8
Pillow 5.3.0
minerl 0.3.0
```
